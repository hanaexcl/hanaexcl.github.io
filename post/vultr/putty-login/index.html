PuTTy連線到Vultr ubuntu Server

July 14, 2020 Hana

1 分鐘

 Twitter

 Facebook

 Reddit

 LinkedIn

 Pinterest

 Email

步驟一 製作金鑰

開啟伺服器的控制網頁，並開啟主控台。

登入主控台，帳密在Overview分頁。

輸入ssh-keygen -t rsa，存檔路徑及密碼都直接Enter。

輸入cd .ssh移動到.SSH資料夾，並輸入mv id_rsa.pub authorized_keys，把id_rsa.pub改名成authorized_keys。

輸入sudo systemctl restart ssh.service，重新啟動ssh service。

步驟二 轉換PPK

用WinSCP連接到伺服器，帳密一樣在Overview分頁。

進入.ssh資料夾。

下載id_rsa檔案。

開啟PuTTygen，Load剛剛下載的id_rsa。(檔案類型需改成All)

點選Save private Key。

會問你是否不使用密碼，點是。

保存剛才產生的PPK檔案。

步驟三 連接SERVER

輸入伺服器的IP。

載入剛才做出來的PPK檔案。

OPEN!

輸入root，然後Enter，就登入進去囉。

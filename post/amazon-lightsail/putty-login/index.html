PuTTy連線到Amazon Lightsail ubuntu Server

July 12, 2020 Hana

1 分鐘

 Twitter

 Facebook

 Reddit

 LinkedIn

 Pinterest

 Email

步驟一 下載PEM

開啟 設定SSH金鑰，下載該主機對應的地區金錀。

步驟二 轉換PPK

開啟PuTTygen，Load剛剛下載的pem檔案。(檔案類型需改成All)

點選Save private Key。

會問你是否不使用密碼，點是。

保存剛才產生的PPK檔案。

步驟三 連接SERVER

輸入伺服器的公有IP。

載入剛才做出來的PPK檔案。

OPEN!

輸入用戶名，然後Enter，就登入進去囉。(用戶名可參考下方)

  • Amazon Linux、openSUSE 和 FreeBSD： ec2-user
  • CentOS AMI： centos
  • Debian AMI： admin or root
  • Fedora AMI： ec2-user or fedora
  • RHEL AMI： ec2-user or root
  • SUSE AMI： ec2-user or root
  • Ubuntu AMI： ubuntu

    • Putty
